## Acerca del programa
Programa escrito en `C`
Programa sencillo que nos permite añadir numeros del árbol baniario asi como tambien eliminar y buscar numeros con un pequeño menu en la terminal, todo esto escrito en lenguaje C.
## Opciones del menu
- Añadir número
- Buscar número
- Eliminar número
- Salir del programa
